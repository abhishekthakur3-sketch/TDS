## Common Layouts

<Tabs>
  <Tab title="Single Column">
    Best for focused content like forms, articles, and detail views. Content is centered with a max-width constraint.

    ```tsx
    <div style={{ maxWidth: '720px', margin: '0 auto', padding: '24px' }}>
      <Header />
      <main>
        {/* Primary content */}
      </main>
      <Footer />
    </div>
    ```

    Use when:
    - Forms and wizards
    - Article or documentation pages
    - Mobile-first layouts
    - Onboarding flows
  </Tab>
  <Tab title="Sidebar + Content">
    Standard layout for dashboards and admin interfaces. Sidebar provides navigation while main area shows content.

    ```tsx
    <div style={{ display: 'flex', minHeight: '100vh' }}>
      <aside style={{ width: '260px', flexShrink: 0 }}>
        <SideNav sections={navItems} />
      </aside>
      <main style={{ flex: 1, padding: '24px' }}>
        {/* Main content */}
      </main>
    </div>
    ```

    Use when:
    - Dashboards and admin panels
    - Settings pages
    - Multi-section applications
  </Tab>
  <Tab title="Split View">
    Two equal or weighted columns for comparison or master-detail patterns.

    ```tsx
    <div style={{ display: 'grid', gridTemplateColumns: '1fr 1fr', gap: '24px' }}>
      <div>{/* Left panel */}</div>
      <div>{/* Right panel */}</div>
    </div>
    ```

    Use when:
    - Comparison views
    - Master-detail interfaces
    - Side-by-side editing
  </Tab>
</Tabs>


## Spacing Guidelines

Use TARMAC spacing tokens to maintain consistent rhythm:

| Token | Value | Use For |
|-------|-------|---------|
| `spacing-4` | 4px | Tight inline spacing |
| `spacing-8` | 8px | Related element gaps |
| `spacing-12` | 12px | Compact component padding |
| `spacing-16` | 16px | Standard component padding |
| `spacing-24` | 24px | Section gaps |
| `spacing-32` | 32px | Major section separation |
| `spacing-48` | 48px | Page-level separation |


## Stacking Order (Z-Index)

| Layer | Z-Index | Elements |
|-------|---------|----------|
| Base | 0 | Page content |
| Sticky | 100 | Sticky headers, floating buttons |
| Dropdown | 200 | Dropdowns, popovers |
| Overlay | 300 | Backdrop overlays |
| Modal | 400 | Modals, dialogs |
| Toast | 500 | Snackbars, notifications |
| Tooltip | 600 | Tooltips |
