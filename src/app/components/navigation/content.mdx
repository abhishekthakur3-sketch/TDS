## Overview

The Navigation system provides mobile navigation patterns including Bottom Nav (tab bar) and Side Nav (drawer menu). Bottom Nav supports 2-5 tabs with two styles and two alignment modes. Side Nav provides single or dual set menu layouts. Together they enable consistent app-level navigation across mobile interfaces.

## Bottom Nav Styles

| Style | Description |
|-------|-------------|
| Primary | Standard bottom nav with underline active indicator |
| Secondary (Rounded) | Rounded pill-style active indicator |


## Bottom Nav Tab Counts

| Count | Tabs | Description |
|-------|------|-------------|
| 2 Tabs | 2 items | Minimal navigation |
| 3 Tabs | 3 items | Standard navigation |
| 4 Tabs | 4 items | Extended navigation |
| 5 Tabs | 5 items | Full navigation |


## Behavior

| Interaction | Result |
|-------------|--------|
| Tap bottom nav tab | Switches to corresponding screen |
| Active tab | Visual indicator (underline or pill) |
| Tap menu icon | Opens side nav drawer |
| Tap side nav item | Navigates to screen, closes drawer |
| Swipe from edge | Opens side nav (gesture) |
| Tap backdrop | Closes side nav |


## Accessibility

| Requirement | Guideline |
|-------------|-----------|
| Role | Use `role="navigation"` on container, `role="tablist"` on bottom nav |
| Active state | `aria-selected="true"` on active tab |
| Labels | Each tab must have a visible label and `aria-label` |
| Keyboard | Arrow keys navigate between tabs |
| Screen reader | Announce tab name and active state |
| Side nav | Use `role="navigation"` with `aria-label="Main menu"` |

```tsx API
interface BottomNavProps {
  style?: 'primary' | 'secondary';
  alignment?: 'vertical' | 'horizontal';
  tabs: BottomNavTab[];
  activeTab: string;
  onTabChange: (tabId: string) => void;
}

interface BottomNavTab {
  id: string;
  label: string;
  icon: ReactNode;
  disabled?: boolean;
  badge?: number;
}

interface SideNavProps {
  variant?: 'single' | 'dual';
  open: boolean;
  onClose: () => void;
  sections: NavSection[];
  activeItem?: string;
  onItemSelect: (itemId: string) => void;
}

interface NavSection {
  title?: string;
  items: NavItem[];
}
```

```tsx Usage
// Bottom nav
<BottomNav
  style="primary"
  alignment="vertical"
  activeTab="home"
  tabs={[
    { id: 'home', label: 'Home', icon: <HomeIcon /> },
    { id: 'orders', label: 'Orders', icon: <OrderIcon /> },
    { id: 'track', label: 'Track', icon: <TrackIcon /> },
    { id: 'profile', label: 'Profile', icon: <ProfileIcon /> },
  ]}
  onTabChange={setActiveTab}
/>

// Side nav
<SideNav
  variant="dual"
  open={showMenu}
  onClose={() => setShowMenu(false)}
  sections={[
    { title: 'Main', items: mainNavItems },
    { title: 'Settings', items: settingsItems },
  ]}
  onItemSelect={handleNavigation}
/>
```
