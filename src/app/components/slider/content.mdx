## Overview

Sliders enable users to adjust values visually by dragging a knob along a track. They provide immediate feedback as the control moves, making them ideal for settings like volume, price ranges, percentages, and thresholds. The system supports both single-value and dual-value (range) configurations.

## Styles

| Style | Track Color | Hex | Use Case |
|-------|------------|-----|----------|
| Black | Black | `#000000` | Default, neutral settings |
| Coal | Dark gray | `#3D445C` | Dark UI contexts |
| Success (Green) | Green | `#1BA86E` | Positive values, active ranges |
| DLV Red | Red | `#ED1B36` | Critical ranges, brand emphasis |

<Info>Use Black for most sliders. Reserve colored styles for contexts where color reinforces semantic meaning.</Info>


## Knob States

| State | Visual Treatment | Interaction |
|-------|-----------------|-------------|
| Default | White knob with colored border, subtle shadow | Draggable |
| Focused | Focus ring around knob (3px outline) | Keyboard focus visible |


## Slider vs Input vs Stepper

| Feature | Slider | Input Field | Stepper |
|---------|--------|-------------|---------|
| Interaction | Drag | Type | Click +/- |
| Precision | Approximate | Exact | Incremental |
| Range selection | Yes (dual) | No | No |
| Best for | Visual adjustment | Exact values | Small increments |
| Feedback | Immediate visual | On submit | On click |


## Accessibility

| Requirement | Implementation |
|-------------|---------------|
| Role | `role="slider"` on the knob element |
| Value | `aria-valuenow`, `aria-valuemin`, `aria-valuemax` |
| Label | `aria-label` or `aria-labelledby` describing the control |
| Keyboard | `←` `→` arrow keys to adjust value by step |
| Large steps | `Page Up` / `Page Down` for larger increments |
| Boundaries | `Home` moves to min, `End` moves to max |
| Focus | Visible focus ring on the knob when focused via keyboard |
| Dual slider | Each knob has its own `role="slider"` with independent values |
| Announcement | Screen readers should announce: "[Label], slider, [value] of [max]" |

<Info>For dual sliders, label each knob distinctly (e.g., "Minimum price" and "Maximum price") so screen readers can differentiate them.</Info>


## Developer Handoff

<CodeGroup>
```tsx API
import { Slider, RangeSlider } from '@tarmac/design-system';

interface SliderProps {
  value?: number;                           // Current value (controlled)
  defaultValue?: number;                    // Initial value (uncontrolled)
  min?: number;                             // Minimum value (default: 0)
  max?: number;                             // Maximum value (default: 100)
  step?: number;                            // Step increment (default: 1)
  style?: 'black' | 'coal' | 'success' | 'red'; // Default: 'black'
  size?: 'small' | 'large';                // Default: 'small'
  disabled?: boolean;                       // Disable interaction
  label?: string;                           // Accessible label
  onChange?: (value: number) => void;       // Value change callback
}

interface RangeSliderProps {
  value?: [number, number];                 // [min, max] values
  defaultValue?: [number, number];          // Initial range
  min?: number;
  max?: number;
  step?: number;
  style?: 'black' | 'coal' | 'success' | 'red';
  size?: 'small' | 'large';
  disabled?: boolean;
  minLabel?: string;                        // Label for min knob
  maxLabel?: string;                        // Label for max knob
  onChange?: (value: [number, number]) => void;
}
```

```tsx Usage
// Single slider
<Slider
  label="Volume"
  defaultValue={60}
  min={0}
  max={100}
  onChange={(val) => setVolume(val)}
/>

// Green success style, large
<Slider
  label="Brightness"
  style="success"
  size="large"
  defaultValue={75}
/>

// Dual range slider
<RangeSlider
  minLabel="Minimum price"
  maxLabel="Maximum price"
  defaultValue={[200, 800]}
  min={0}
  max={1000}
  step={50}
  onChange={([min, max]) => setPriceRange(min, max)}
/>

// With input field pairing
<div style={{ display: 'flex', alignItems: 'center', gap: '16px' }}>
  <Slider
    label="Threshold"
    value={threshold}
    onChange={setThreshold}
    style="red"
  />
  <Input value={threshold} onChange={setThreshold} width="60px" />
</div>
```

```tsx ARIA
// Single slider
<div role="group" aria-labelledby="volume-label">
  <label id="volume-label">Volume</label>
  <input
    type="range"
    role="slider"
    aria-valuenow={60}
    aria-valuemin={0}
    aria-valuemax={100}
    aria-label="Volume"
  />
</div>

// Dual range slider
<div role="group" aria-label="Price range">
  <input

    aria-valuenow={200}
    aria-valuemin={0}
    aria-valuemax={800}
```
