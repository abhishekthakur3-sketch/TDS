## Overview

Scrollbars allow users to navigate through content that extends beyond the visible viewport. They provide a visual indicator of the current viewing position within scrollable content. The component consists of a track (background container) and a thumb (draggable handle) that reflects the proportion of visible content relative to total content.

## Styles

| Style | Color | Emphasis | Use When |
|-------|-------|----------|----------|
| Grey | Neutral grey | Low | Subtle or neutral UI surfaces |
| Coal | `#3D445C` | Medium | Structured containers, panels |
| Black | `#000000` | High | Dark or high-emphasis surfaces |


## Sizes

| Size | Thickness | Track Length | Use When |
|------|-----------|-------------|----------|
| Large | 6px | Varies | Primary content containers, improved visibility |
| Small | 4px | Varies | Compact layouts, dense tables, panels, side drawers |


## Behavior

| Interaction | Result |
|-------------|--------|
| Drag thumb | Scrolls content proportionally |
| Mouse wheel / trackpad | Scrolls content in the overflow direction |
| Click inside track | Jumps scroll position to clicked area |
| Content changes | Thumb size updates to reflect content ratio |
| No overflow | Scrollbar is hidden |


## Accessibility

| Requirement | Guideline |
|-------------|-----------|
| Keyboard | Scroll containers must support arrow keys and Page Up/Down |
| Focus | Focus indicators must remain visible within scrollable areas |
| Contrast | Scrollbar thumb must have sufficient contrast against the track |
| Overflow indication | Provide visual cues that content is scrollable |
| Native behavior | Preserve native scroll semantics for assistive technologies |


## Developer Handoff

<CodeGroup>
```tsx API
interface ScrollbarProps {
  style?: 'grey' | 'coal' | 'black';
  orientation?: 'vertical' | 'horizontal';
  size?: 'lg' | 'sm';
  className?: string;
}
```

```tsx Usage
// Default vertical scrollbar
<Scrollbar style="grey" orientation="vertical" size="lg" />

// Horizontal small scrollbar
<Scrollbar style="coal" orientation="horizontal" size="sm" />

// High contrast for dark surfaces
<Scrollbar style="black" orientation="vertical" size="lg" />

// CSS custom scrollbar styling
.scrollable-container {
  overflow-y: auto;
  scrollbar-width: thin;
  scrollbar-color: var(--scroll-grey-thumb) transparent;
}

.scrollable-container::-webkit-scrollbar {
  width: 6px;
}

.scrollable-container::-webkit-scrollbar-thumb {
  background: var(--scroll-grey-thumb);
  border-radius: 3px;
}
```
