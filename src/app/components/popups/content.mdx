## Overview

Popups are overlay containers that display contextual content above the main interface. They consist of a Header, Content Container (slot-based), optional Snackbar, optional Tab Group, and Footer with action buttons. The system supports five sizes from XSmall (mobile) to XLarge (wide desktop) and includes dedicated header and footer sub-components.

## Sub-Components

| Component | Variants | Description |
|-----------|----------|-------------|
| Popup Header | Large (82px), Medium (80px), Small (64px) | Title, subtitle, close button, optional actions |
| Popup Footer | Large (60px), Medium (60px), Small (56px) | Action buttons (primary, secondary, tertiary) |
| Content Container | Slot-based | Flexible area for custom content |
| Snackbar | Optional | Contextual information bar below header |
| Tab Group | Optional | Tab navigation within the popup |


## Behavior

| Interaction | Result |
|-------------|--------|
| Popup opens | Overlay backdrop appears, focus trapped inside |
| Click close button | Closes popup |
| Click primary action | Executes action, may close popup |
| Click secondary action | Cancels or alternative action |
| Click backdrop | Closes popup (if dismissible) |
| Press Escape | Closes popup |
| Tab key | Cycles focus within popup |
| Scroll | Content area scrolls independently |


## Accessibility

| Requirement | Guideline |
|-------------|-----------|
| Role | Use `role="dialog"` with `aria-modal="true"` |
| Label | Provide `aria-labelledby` pointing to the header title |
| Focus trap | Focus must be trapped inside the popup while open |
| Keyboard | Escape closes, Tab cycles through elements |
| Screen reader | Announce popup title on open |
| Return focus | Return focus to trigger element on close |


## Developer Handoff

<CodeGroup>
```tsx API
interface PopupProps {
  size?: 'xs' | 'sm' | 'md' | 'lg' | 'xl';
  open: boolean;
  onClose: () => void;
  title: string;
  subtitle?: string;
  snackbar?: ReactNode;
  tabs?: TabProps[];
  children: ReactNode;
  primaryAction?: { label: string; onClick: () => void };
  secondaryAction?: { label: string; onClick: () => void };
  tertiaryAction?: { label: string; onClick: () => void };
  dismissible?: boolean;
}
```

```tsx Usage
// Medium popup with form
<Popup
  size="md"
  open={showPopup}
  onClose={() => setShowPopup(false)}
  title="Create Order"
  subtitle="Fill in the order details"
  primaryAction={{ label: 'Submit', onClick: handleSubmit }}
  secondaryAction={{ label: 'Cancel', onClick: () => setShowPopup(false) }}
>
  <OrderForm />
</Popup>

// Large popup with tabs
<Popup
  size="lg"
  open={showDetails}
  onClose={() => setShowDetails(false)}
  title="Order Details"
  tabs={[
    { label: 'Summary', content: <Summary /> },
    { label: 'Timeline', content: <Timeline /> },
  ]}
  primaryAction={{ label: 'Close', onClick: () => setShowDetails(false) }}
>
  {activeTabContent}
</Popup>

// XSmall mobile popup
<Popup
  size="xs"
  open={showMobile}
  onClose={() => setShowMobile(false)}
  title="Confirm"
  primaryAction={{ label: 'OK', onClick: handleOk }}
>
  <p>Are you sure?</p>
</Popup>
```
