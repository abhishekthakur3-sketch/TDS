## Overview

Pagination is a navigation pattern for browsing large datasets across multiple pages. It combines page number cells, navigation controls (Previous/Next), and contextual text (e.g., "Showing 200 results") into a structured bar. The system is built from four foundational elements that assemble into a complete pagination experience.

## Text Group Variants

| Variant | Example | Description |
|---------|---------|-------------|
| Single | Showing 200 | Item count only |
| Dual | Showing 200 results | Item count + label |
| Triple | Showing 200 \| Page 3 | Count + page info |
| Quad | Showing 200 \| Page 3 of 8 | Full context |


## Pagination vs Infinite Scroll vs Load More

| Feature | Pagination | Infinite Scroll | Load More |
|---------|-----------|----------------|-----------|
| Navigation | Jump to any page | Continuous scroll | Append on click |
| URL support | Yes (page param) | Difficult | Difficult |
| Performance | Predictable | Can degrade | Moderate |
| Best for | Tables, search results | Social feeds, galleries | Content listings |
| SEO | Good | Poor | Moderate |


## Accessibility

| Requirement | Implementation |
|-------------|---------------|
| Landmark | Wrap in `<nav aria-label="Pagination">` |
| Current page | `aria-current="page"` on the active page button |
| Labels | Each page button: `aria-label="Page 3"` or `aria-label="Go to page 3"` |
| Disabled | `aria-disabled="true"` on Previous (page 1) and Next (last page) |
| Keyboard | `Tab` to navigate between controls, `Enter` to activate |
| Announcement | Screen readers should announce "Page 3 of 8" on navigation |
| List structure | Use `<ol>` for page numbers to convey sequence |

<Info>Always use `aria-current="page"` on the active page number so screen readers announce which page the user is currently viewing.</Info>

<CodeGroup>
```tsx API
import { Pagination } from '@tarmac/design-system';

interface PaginationProps {
  currentPage: number;                      // Current active page (1-indexed)
  totalPages: number;                       // Total number of pages
  totalItems?: number;                      // Total items (for context text)
  size?: 'large' | 'medium' | 'small';     // Default: 'medium'
  style?: 'black' | 'blue' | 'red';        // Default: 'black'
  showContext?: boolean;                    // Show "Showing X results"
  showPageInfo?: boolean;                   // Show "Page X of Y"
  onPageChange: (page: number) => void;    // Page change callback
}
```

```tsx Usage
// Basic pagination
<Pagination
  currentPage={3}
  totalPages={8}
  onPageChange={(page) => setCurrentPage(page)}
/>
```
