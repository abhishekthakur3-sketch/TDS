## Core Motion Patterns


  <Accordion title="The Runway — Smooth Acceleration">
    Like aircraft on a runway, elements accelerate and decelerate smoothly.

    Easing: `cubic-bezier(0.4, 0.0, 0.2, 1)`

    Use for: Page transitions, modal appearances, large element movements.
  </Accordion>
  <Accordion title="The Bounce — Subtle Spring">
    Subtle spring-like motion that adds personality without distraction.

    Easing: `cubic-bezier(0.68, -0.55, 0.265, 1.55)`

    Use for: Button presses, toggle switches, success confirmations.
  </Accordion>
  <Accordion title="The Fade — Gentle Opacity">
    Gentle opacity changes that don't draw unnecessary attention.

    Easing: `ease-in-out`

    Use for: Tooltips, hover states, loading states.
  </Accordion>
  <Accordion title="The Slide — Directional Movement">
    Directional movement that shows spatial relationships.

    Easing: `cubic-bezier(0.4, 0.0, 0.2, 1)`

    Use for: Drawers, panels, dropdown menus, carousel navigation.
  </Accordion>

## Testing Motion

| Check | How |
|-------|-----|
| Reduced motion | Enable `prefers-reduced-motion` in OS settings |
| Slower devices | Test on low-end hardware or throttle CPU |
| Interaction blocking | Verify animations don't block user input |
| Cross-browser | Test in Chrome, Firefox, Safari, Edge |
